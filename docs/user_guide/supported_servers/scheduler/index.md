# Scheduler MCP Servers

Scheduler servers manage job scheduling and execution. They help coordinate computational workloads across resources.

## Available Scheduler Servers

| Server Name           | Description                                  | Command to Start      |
| --------------------- | -------------------------------------------- | --------------------- |
| [Flux](./flux.md)     | Flux job scheduler and execution engine      | `mada-mcp-flux`       |
| [Slurm](./slurm.md)   | Slurm workload manager for job scheduling    | `mada-mcp-slurm`      |

Refer to individual server pages for setup and usage instructions. See the [Supported MCP Servers](../index.md) page for a complete list of supported servers.
