# Workflow MCP Servers

Workflow servers provide study construction and execution tools. These servers support templated study construction, workflow execution, status checking, and more.

## Available Workflow Servers

| Server Name                   | Description                                  | Command to Start      |
| ----------------------------- | -------------------------------------------- | --------------------- |
| [Maestro Command Executor](./maestro.md)   | Maestro workflow orchestration tool command execution   | `mada-mcp-maestro-command-executor`  |
| [WEAVE Study Constructor](../../../developer_guide/server_creation/weave_study_servers.md) | An abstract class *only intended for developer usage* that can be inherited from to construct YAML files that WEAVE tools can ingest | N/A |

Check each server’s documentation for instructions and examples. See the [Supported MCP Servers](../index.md) page for a complete list of supported servers.
