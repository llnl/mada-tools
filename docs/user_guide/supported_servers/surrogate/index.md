# Surrogate MCP Servers

Surrogate servers provide analysis and visualization tools. These servers support data analysis, post-processing, and visualization of simulation results.

## Available Surrogate Servers

| Server Name                   | Description                                  | Command to Start      |
| ----------------------------- | -------------------------------------------- | --------------------- |
| [Professor](./professor.md)   | Professor analysis and visualization tools   | `mada-mcp-professor`  |

Check each server’s documentation for instructions and examples. See the [Supported MCP Servers](../index.md) page for a complete list of supported servers.
