# Simulation MCP Servers

Simulation servers run computational codes for scientific and engineering simulations. These servers are designed to execute models using prepared geometries and input data.

## Available Simulation Servers

| Server Name                   | Description                             | Command to Start      |
| ----------------------------- | --------------------------------------- | --------------------- |
| [Vertex-CFD](./vertex_cfd.md) | Vertex-CFD simulation application tools | `mada-mcp-vertex-cfd` |

See server-specific documentation for configuration and usage. See the [Supported MCP Servers](../index.md) page for a complete list of supported servers.
